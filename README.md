# Student Management System 학생 성적 관리 시스템
<img width="62%" alt="스크린샷 2024-07-05 오후 7 29 26" src="https://github.com/rudcks8389/project_ams/assets/158795491/193a30d5-ac77-48a1-bdd9-71a806fff145">

## 자바스크립트로 만든 개인 프로젝트입니다.
과목별 점수, 총점, 평균, 순위 등 한 눈에 볼 수 있도록 표 형식으로 제작했습니다.

<br/>
<br/>

## 🗓 프로젝트 기간
2024년 01월 10일 ~ 2024년 01월 12일

<br/>
<br/>

## ✨ 주요기능
1. 입출금계좌생성
2. 대출계좌생성
3. 계좌목록
4. 예금
5. 출금
6. 계좌검색
7. 계좌삭제

<br/>
<br/>

## 🛠️ 개발환경
IntelliJ

<br/>
<br/>

## 🕹️ 사용 기술
Java
